# Anti-trial Abuse MVP - 实现路线图

阶段：3/7 闪电战实现策略
日期：2026-03-23

## MVP 命名：TrialGuard API

一个简单的API服务，检测用户是否为白漂账户。

## 核心功能（只写必要功能）

### 必须有（3天内完成）
1. **邮箱指纹检测** - 检测邮箱是否来自已知滥用域名（可丢弃邮箱、+suffix等）
2. **设备指纹检测** - 浏览器指纹 + IP检测
3. **风险评分API** - 输入用户信息，返回0-100风险分数

### 暂时不做
- 复杂的行为分析
- 数据库存储
- 管理后台

## 技术栈

- 后端：Node.js + Express（或 Python FastAPI）
- 数据源：公开的可疑邮箱域名列表
- 部署：Render / Railway 免费层

## 里程碑

| 时间 | 任务 | 交付物 |
|------|------|--------|
| 第1天 | 核心API开发 | trialguard-api 代码 |

## 付费点设计

- 免费层：50次检测/月
- 付费层：$29/月，无限检测
- 目标客户：AI SaaS创始人
